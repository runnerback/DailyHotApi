# Prompt: Perler Beads Effect Image

## English Prompt

```
Create a pixel art illustration in the style of Perler beads (fuse beads / iron beads).

Subject: [SUBJECT]

Style requirements:
- Perler bead aesthetic: each "pixel" should look like a small, round, slightly raised plastic bead with a visible center hole, arranged on a flat pegboard grid
- Use a limited but vibrant color palette typical of Perler bead kits (solid colors, no gradients within individual beads)
- The beads should have subtle 3D volume — a slight dome shape with soft specular highlights, as if made of glossy plastic
- Visible grid alignment: all beads snap to a uniform square grid, no anti-aliasing or sub-pixel blending between bead edges

Composition:
- The design must fill the entire canvas edge-to-edge with no background margin, border, or empty pegboard showing
- Frame the subject tightly so the bead artwork occupies 100% of the image area
- Maintain clear, recognizable shapes — simplify complex details into chunky pixel blocks that read well at bead scale

Rendering:
- Flat, even lighting from above to showcase bead colors accurately
- No perspective distortion — render as a perfectly orthographic / straight-on front view
- Clean, crisp edges between beads; no motion blur or depth-of-field

Image size: 1024 x 1024 pixels
```

---

## 中文说明

> 用途：根据新闻热点事件，生成拼豆风格的视觉效果图（非图纸）。
> 后续流程：效果图 → 独立转换工具 → 拼豆图纸。

### 使用方法

1. 将 `[SUBJECT]` 替换为具体的视觉主题描述，例如：
   - `A giant panda sitting among bamboo, celebrating China's space station completion`
   - `A golden trophy cup with the text "2026" for year-end awards season`
2. 主题描述应**视觉化**：将抽象新闻事件转化为可画的具体画面
3. 生成的效果图将通过转换工具生成实际拼豆图纸，因此画面需保持**像素块感清晰、色彩分明、轮廓简洁**

### Prompt 要点

| 要素 | 说明 |
|------|------|
| 风格 | 拼豆美学 — 每个像素是圆形塑料珠，有中心孔、微凸光泽 |
| 色彩 | 拼豆标准色板，纯色填充，无渐变 |
| 网格 | 严格方格对齐，珠子间无抗锯齿过渡 |
| 构图 | 填满整个画幅，无边距、无背景留白 |
| 视角 | 正交正面视角，无透视变形 |
| 光照 | 均匀顶光，真实展示珠子颜色 |
| 尺寸 | 1024 x 1024 像素 |
